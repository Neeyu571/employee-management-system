# Employee Management System

A Full Stack Employee Management System built using React.js, Spring Boot, and MySQL.

## Technologies Used

Frontend:
- React.js
- React Router
- Axios
- Bootstrap

Backend:
- Spring Boot
- Spring Data JPA
- REST API

Database:
- MySQL

## Features

- Add Employee
- View Employee List
- Update Employee
- Delete Employee

employee-management-system

├── Backend
│   └── springbootrestapi
│
└── Frontend
